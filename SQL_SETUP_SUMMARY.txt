## Database Schema

### USERS Table
```
ID (NUMBER) - Primary Key, Auto-increment
USERNAME (VARCHAR2(100)) - Unique, Not Null
PASSWORD (VARCHAR2(255)) - Not Null (BCrypt hash)
```

### EXPENSES Table
```
ID (NUMBER) - Primary Key, Auto-increment
CATEGORY (VARCHAR2(100)) - Not Null
AMOUNT (DECIMAL(10,2)) - Not Null
COMMENTS (VARCHAR2(500)) - Optional
USERNAME (VARCHAR2(100)) - Foreign Key to USERS.USERNAME
CREATED_AT (TIMESTAMP) - Auto-set on insert
UPDATED_AT (TIMESTAMP) - Auto-set on insert/update
```

## Quick Start

### Option 1: Manual SQL Script (Recommended)
```
1. Open sql/init-schema.sql in Oracle SQL Developer
2. Click Run Script (F5)
3. Start the application: mvn spring-boot:run
```

### Option 2: Auto-Initialization on Startup
```
1. Copy sql/data.sql to src/main/resources/data.sql
2. Change application.properties: spring.jpa.hibernate.ddl-auto=create
3. Tables and data auto-create on first run
```
